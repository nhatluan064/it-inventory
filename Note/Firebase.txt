rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    // Chỉ cho phép người dùng đã xác thực
    // truy cập vào bộ sưu tập "users" của chính họ.
    match /users/{userId}/{document=**} {
      allow read, write: if request.auth != null && request.auth.uid == userId;
    }
  }
}

Hosting (Create Hosting) la xong


npm install firebase (Nếu chưa cài đặt gói firebase cho dự án mới nhất)
npm install -g firebase-tools (Cài bộ công cụ của firebasetools )
firebase login (Khi cài Firebase Tool thì mới sử dụng duoc chuc nang Firebase Login)
firebase init (Firebase init cung the, no dung de tao hosting)
npm run build (Sau khi chinh sua cau hinh he thong xong, ta cho phep build lai de chuan bi leploy)
firebase deploy (Deploy len Server Firebase Database va Hosting)
